# EduTech Learning Management System

A modern, responsive Learning Management System (LMS) built with HTML, CSS, JavaScript, and Tailwind CSS. This comprehensive educational platform provides course management, student enrollment, assignment tracking, discussion forums, and certificate generation.

## 🌟 Features

### Core LMS Features
- **Course Creation and Management** - Comprehensive course builder with content management
- **Student Enrollment System** - Easy course enrollment and progress tracking
- **Video Streaming and Content Delivery** - High-quality video content support
- **Assignment and Quiz Modules** - Interactive assignments with automated grading
- **Progress Tracking and Analytics** - Advanced performance analytics and reporting
- **Discussion Forums** - Community-driven learning with peer interaction
- **Certificate Generation** - Automated certificate issuance upon completion
- **Payment Integration** - Flexible pricing and payment options

### User Interface Features
- **Modern, Clean Design** - Minimalistic layout with generous white space
- **Responsive Design** - Optimized for desktop, tablet, and mobile devices
- **Sticky Navigation** - Transparent navbar that becomes solid on scroll
- **Interactive Components** - Hover effects, animations, and smooth transitions
- **Accessibility Features** - Focus outlines and keyboard navigation support
- **Dark Mode Support** - Theme switching capability

### Dashboard Features
- **Overview Dashboard** - Comprehensive learning analytics and progress tracking
- **Course Management** - View and manage enrolled courses with progress indicators
- **Assignment Tracking** - Complete assignment management with due dates and grades
- **Discussion Forums** - Categorized forums with search and filtering
- **Grade Reports** - Detailed grade breakdowns and performance analytics
- **Certificate Gallery** - Digital certificate management and verification
- **Profile Management** - Comprehensive user profile and settings

## 🚀 Getting Started

### Prerequisites
- Modern web browser (Chrome, Firefox, Safari, Edge)
- Basic understanding of HTML, CSS, and JavaScript
- Text editor or IDE for customization

### Installation

1. **Clone or Download** the project files to your local machine
2. **Open the project folder** in your preferred code editor
3. **Launch a local server** (recommended) or open `index.html` directly in your browser

### Using a Local Server (Recommended)

#### Option 1: Python
```bash
# Python 3
python -m http.server 8000

# Python 2
python -m SimpleHTTPServer 8000
```

#### Option 2: Node.js
```bash
# Install global server
npm install -g http-server

# Run server
http-server
```

#### Option 3: PHP
```bash
php -S localhost:8000
```

#### Option 4: VS Code Live Server Extension
- Install the "Live Server" extension in VS Code
- Right-click on `index.html` and select "Open with Live Server"

## 🛠️ Customization

### Styling
The project uses Tailwind CSS for styling. To customize:

1. **Colors**: Modify color classes in HTML or add custom CSS
2. **Layout**: Adjust container sizes and spacing using Tailwind classes
3. **Typography**: Change font families and sizes in the CSS files
4. **Components**: Customize individual components by editing their CSS

### Functionality
JavaScript functionality is modular and can be extended:

1. **Navigation**: Modify `scripts/main.js` for landing page interactions
2. **Dashboard**: Edit `scripts/dashboard.js` for dashboard-specific features
3. **Forms**: Add form validation and submission handling
4. **API Integration**: Connect to backend services for dynamic data

### Content
Customize content by editing the HTML files:

1. **Text Content**: Update headings, descriptions, and labels
2. **Images**: Replace placeholder images with actual content
3. **Course Data**: Modify course information and pricing
4. **User Information**: Update sample user data and statistics
